Give free gifts to your WooCommerce customers

== Description ==

SMNTCS WooCommerce Free Gift is a lightweight plugin to give away free gifts to your WooCommerce customers

== Installation ==

1. Upload the plugin to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to 'Apperance' Â» 'Customize'
4. Enable free gifts
5. Adjust settings based on your needs
